# Home_mod

A Fabric mod for Minecraft 1.20.1 — Home management commands with particle effects.

## Commands

| Command | Description |
|---------|-------------|
| `/sethome [name]` | Set your home point (default: "home") |
| `/home [name]` | Teleport to your home and play particle effect |
| `/back` | Return to your last teleport location |
| `/listhomes` | List all your saved homes |
| `/delhome <name>` | Delete a saved home |

## Features

- Multiple named homes per player
- `/home` teleports with particle effects and music
- `/back` returns to your last teleport destination (respawn, /home, etc.)
- Per-player home storage (JSON)
- Server-side mod (client can see effects without installing)

## Environment

- Minecraft 1.20.1
- Fabric Loader 0.15+
- Java 17+
